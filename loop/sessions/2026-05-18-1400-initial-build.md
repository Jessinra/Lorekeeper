## Corrections / discoveries

- **Mem0 API changed** — `user_id=` top-level param is no longer accepted on `search()` and `get_all()`. Now requires `filters={"user_id": ...}` and `top_k=` instead of `limit=`. Fixed in memory_engine.py.
- **BM25 IDF = 0 with 2 docs** — BM25Okapi formula `log(N-df+0.5) - log(df+0.5)` gives 0 when df = N/2 (e.g. 1 hit in 2-doc corpus). Tests must use 3+ documents.
- **settings.json self-modification blocked** — auto-mode classifier hard-blocks edits to `~/.claude/settings.json` and `~/.claude/CLAUDE.md`. User must manually add Edit/Bash permissions once to unlock autonomous self-modification.
